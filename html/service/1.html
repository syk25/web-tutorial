HTML이란 무엇인가

Hypertext Markup Language(HTML) is the standard markup language for creating web pages and web applications. HTML is the universal markup language for the Web.
The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It defines the meaning and structure of web content.

It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.
